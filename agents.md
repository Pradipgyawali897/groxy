# Organization Hierarchy

## EXECUTIVE LAYER
- **CEO AGENT**: Global orchestration, architecture governance, workflow supervision, and task delegation.

## ENGINEERING DIVISIONS
- **CTO AGENT**: Architecture enforcement, performance governance, and bundle optimization.
- **SECURITY DIRECTOR AGENT**: Supabase auth, JWT lifecycle, RBAC, and secure session persistence.
- **MARKET DIRECTOR AGENT**: Inventory orchestration, secondhand workflows, and search indexing.
- **CHECKOUT DIRECTOR AGENT**: Payment orchestration and transaction integrity.
- **SEO DIRECTOR AGENT**: Metadata optimization, semantic HTML, and structured data.
- **UI DIRECTOR AGENT**: Minimalist design system and component simplification.

## SPAWNABLE WORKER AGENTS
- **DeadCodeCleanerWorker**: Identifies and removes unused code/UI.
- **BundleOptimizerWorker**: Analyzes and optimizes JS/CSS bundles.
- **ReservationWorker**: Handles atomic reservation locks for unique items.
- **SearchIndexWorker**: Manages catalog indexing.
- **SEOAuditWorker**: Validates SEO compliance.
- **HydrationFixWorker**: Detects and fixes React hydration mismatches.
