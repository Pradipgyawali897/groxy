# Functional Gap Registry

This registry tracks prioritized missing functionality for the platform.

## Phase 1: High Priority (Market Trust & Platform Health)
- [ ] **Book Comparison Tool**: Enables buyers to compare multiple copies of the same book based on condition, price, and seller trust.
- [ ] **Moderation Queue (Admin)**: Centralized interface for moderating book listings and seller disputes.
- [ ] **Search Results (Advanced)**: Dynamic filtering UI for faceted search.

## Phase 2: Growth & Monetization
- [ ] **Listing Creation Wizard**: Step-by-step merchant listing flow.
- [ ] **Seller Earnings Dashboard**: Transaction visibility for sellers.
- [ ] **Order Fulfillment Flow**: Merchant tools for managing active orders.

## Phase 3: Infrastructure & Reliability
- [ ] **Activity Log Page**: Secure, read-only audit log for sensitive system events.
- [ ] **System Health Monitoring**: Real-time observability dashboard for infra agents.
